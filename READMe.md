---

# 📖 Module Overview

Application Development Security focuses on building secure web applications using modern development practices.

Throughout this semester, you will progressively develop a secure full-stack application while learning how to:

- Build RESTful APIs using Node.js and Express
- Develop modern React applications
- Secure applications using authentication and authorisation
- Work with MongoDB databases
- Test backend and frontend applications
- Containerise applications using Docker
- Automate builds using GitHub Actions
- Apply DevSecOps principles

This is a **highly practical module**, so you are expected to code every week.
