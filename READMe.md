# 🛡️ INSY7314 – Application Development Security

Welcome to the official repository for **INSY7314 – Application Development Security**.

This repository contains lecture material, practical activities, sample code, weekly resources, and project files used throughout the semester.

---

# 📖 Module Overview

Application Development Security focuses on designing, building, testing, and deploying secure modern web applications using current industry practices.

Throughout this semester you will learn how to:

* Build RESTful APIs using Node.js and Express
* Develop interactive React applications
* Secure applications using authentication and authorisation
* Work with MongoDB databases
* Test frontend and backend applications
* Containerise applications using Docker
* Automate development using GitHub Actions
* Apply DevSecOps principles
* Deploy and monitor applications

This is a **highly practical module**, meaning you'll be writing code almost every week.

---

# 🎮 Semester Project — GameVault

Instead of creating lots of unrelated projects, we'll build **one application throughout the semester.**

Our semester-long project is called **GameVault**.

GameVault is a secure full-stack web application where users can:

* 🎮 Browse video games
* ⭐ Review and rate games
* 📚 Manage their own game collection
* ❤️ Create wishlists
* 👤 Register and log in securely
* 🛡️ Access features based on user roles

Every Learning Unit introduces new technologies and security concepts that expand the application.

By the end of the semester, GameVault will be a complete MERN application built using modern software engineering practices.

> 💡 **Important:** GameVault is a learning project. Your assessments will require you to apply these concepts independently to a different problem domain.

# 🕹️ GameVault Progression

## 📚 Learning Unit 1 — Foundations

Build the foundation of a secure Node.js application.

You'll learn:

* Node.js fundamentals
* Express
* API routes
* JSON
* Environment variables
* Input validation
* Basic security concepts

---

## 🔧 Learning Unit 2 — Secure Backend Development

Expand the backend into a structured REST API.

New features include:

* CRUD operations
* Express routing
* Controllers
* Middleware
* HTTPS
* Helmet
* Content Security Policy (CSP)
* CORS
* API testing using Postman

---

## 🔐 Learning Unit 3 — Authentication & Data Security

Secure the application and introduce users.

Features include:

* User registration
* Login
* Password hashing
* JSON Web Tokens (JWT)
* Role-Based Access Control (RBAC)
* Protected routes
* Rate limiting
* MongoDB integration

---

## 💻 Learning Unit 4 — Frontend Development

Build the React frontend.

Features include:

* User login and registration
* Browse games
* View game details
* Personal collections
* Reviews
* User profiles
* Admin dashboard

---

## 🧪 Learning Unit 5 — Testing & Containerisation

Improve quality and reliability.

Topics include:

* Automated testing
* Postman & Newman
* React testing
* ESLint
* Docker
* Docker Compose

---

## ⚙️ Learning Unit 6 — DevSecOps & Deployment

Prepare the application for production.

You'll implement:

* GitHub Actions
* CI/CD pipelines
* Static code analysis
* Automated testing
* Logging
* Monitoring
* Secure deployment
