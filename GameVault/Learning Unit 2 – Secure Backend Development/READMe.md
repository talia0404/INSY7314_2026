### 🔧 Learning Unit 2 – Secure Backend Development

We'll transform our basic server into a structured backend API.

New features include:

* CRUD operations for games
* Express routing
* Controllers
* Middleware
* HTTPS
* Helmet
* Content Security Policy (CSP)
* CORS
* API testing with Postman

At this stage, GameVault will become a professional-looking backend application.
